# 💱 Currency Converter – Real-Time Exchange App

🚀 **Live Demo:**
🔗 [Live Demo](https://webdevzone.in)

🚀 A modern and responsive currency converter web app that provides real-time exchange rates using an API.
Built to practice **JavaScript async handling**, **API integration**, and **responsive UI design**.

---

## 📌 Tech Stack

- 🧱 HTML5
- 🎨 CSS3
- ⚙️ JavaScript

---

## ✨ Features

- 💱 Real-time currency conversion
- 🌍 Supports multiple international currencies
- ⚡ Fast API-based exchange rate fetching
- 🔄 Swap currencies instantly
- 📱 Fully responsive design (mobile + desktop)
- 🎯 Clean and minimal UI
- ❌ Input validation with error handling

## 🚀 Key Highlights

## 🛠️ Installation

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone - https://github.com/webdev-desktop/Currency-Converter.git

# 2. Go into the project folder
cd Currency-Converter

# 3. Open index.html in your browser
```

---

## ▶️ Usage

1. Enter the amount 💰
2. Select **From Currency**
3. Select **To Currency**
4. Click on **Get Exchange Rate**
5. View the converted result instantly 🚀
6. Use 🔄 icon to swap currencies
